Rubik's Cube White Side Solver - API

This is the API server for the Rubik's Cube White Side Solver educational application. The React frontend should be running separately.

Available Endpoints:

POST /api/start

Start a new learning session

GET /api/module/{module_id}

Get learning module content by ID

POST /api/module/{module_id}/complete

Mark a module as complete and save user answers

GET /api/quiz/{question_id}

Get quiz question by ID

POST /api/quiz/{question_id}/answer

Submit an answer for a quiz question

GET /api/results

Get quiz results and score

POST /api/reset-quiz

Reset quiz answers to retake the quiz
